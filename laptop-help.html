15 Common Laptop & PC Solutions

Try these DIY fixes from Shree Sanwariya Computers before visiting the shop.

1. Laptop Not Turning On

Hold the power button for 30 seconds without the charger. Plug it back in and try again.

2. Extremely Slow Performance

Check Task Manager (Ctrl+Alt+Del) for high Disk usage. Most slow PCs in Udaipur need an SSD Upgrade.

3. Blue Screen (BSOD)

Restart and unplug USB devices. If it persists, it’s a RAM or driver failure.

4. WiFi Not Connecting

Toggle Flight Mode ON/OFF. If missing, check if the "WiFi" key on your keyboard is pressed.

5. Battery Not Charging

Check for dust in the charging port. Try a different wall socket or power adapter.

6. Loud Fan Noise / Heating

Use your laptop on a hard surface. If it’s over 2 years old, it needs Internal Thermal Cleaning.

7. Keyboard Keys Not Working

Go to Device Manager and Uninstall the Keyboard driver, then restart the laptop.

8. Auto-Restarting PC

Usually caused by overheating or a failing SMPS (Power Supply). Clean the processor fan.

9. Browser Redirecting/Ads

Your PC has malware. Reset Chrome settings and uninstall unknown "Extensions."

10. Screen is Flickering

Update your Display Drivers. If flickering happens when moving the lid, the LCD Cable is damaged.

11. No Sound Output

Right-click the Speaker icon > Troubleshoot Sound. Ensure the correct "Output Device" is selected.

12. USB Port Not Working

Disable "USB selective suspend" in Power Options. Test with a different mouse or pen drive.

13. Windows Update Stuck

Run the Windows Update Troubleshooter in Settings. Ensure you have at least 20GB free space.

14. Forgotten Login Password

Use "Reset Password" via Microsoft account. If local, you may need a Windows Re-installation.

15. Touchpad Not Working

Check if the Function key (e.g., F6 or F9) has disabled the touchpad. Restart the system.
